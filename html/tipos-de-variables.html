📊 Clasificación y Captura de Datos

La Variable Estadística es el objeto de nuestro estudio. Se divide según la naturaleza del dato:

✨ Cualitativa

Valores no numéricos (texto/categorías).
Ej: Ciudad, Red Social.

🔢 Cuantitativa

Valores numéricos.

  • Discreta: Valores aislados (nº likes).
  • Continua: Intervalos (peso, tiempo).

Como auditores, podemos recolectar la información de dos formas:

🎤
Obtención Directa

Preguntando o midiendo directamente al individuo (Encuestas, cuestionarios, observación).

📂
Obtención Indirecta

Usando fuentes ya existentes (Bases de datos, prensa, informes institucionales, API externas).
