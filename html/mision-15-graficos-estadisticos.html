Misión #15: Gráficos estadísticos

Introducción

📊 Gráficos: La Cara de tus Datos

Nadie quiere leer una tabla con 500 filas. Los gráficos son los que venden tu trabajo: transforman números fríos en historias visuales que cualquier cliente o marca puede entender en un segundo.

Dependiendo de si tus datos son categorías (colores, países) o números (likes, ventas), usaremos un gráfico diferente para presentarlos.

Características

Diagrama de Barras

Es el gráfico estándar para comparar cantidades entre categorías discretas.

  • Variable: Cualitativas (nombres) o Cuantitativas discretas (números enteros).
  • Forma: Barras rectangulares separadas entre sí.
  • Ejes: El eje X muestra las categorías; el eje Y muestra la frecuencia ($f_i$).

Histograma

Se usa para representar la distribución de datos que fluyen sin interrupción.

  • Variable: Cuantitativas continuas agrupadas en intervalos.
  • Forma: Barras rectangulares unidas (sin espacio entre ellas).
  • Significado: El área de cada barra es proporcional a la frecuencia del intervalo.

Diagrama de Sectores

Ideal para visualizar la composición de un "todo" en partes porcentuales.

  • Variable: Cualitativas con pocas categorías (máximo 5 o 6).
  • Forma: Un círculo dividido en cuñas o "quesitos".
  • Cálculo: El ángulo de cada sector es $\alpha = \frac{360^\circ \cdot f_i}{N}$.
