Diagrama de sectores

Introducción

🎯 El Núcleo de Reparto

Si el histograma es un radar de densidad, el Diagrama de Sectores es el visor de dominancia. Su misión es simple pero crítica: dividir una unidad total en segmentos proporcionales para revelar qué parte del todo pertenece a cada categoría.

En Mirror Corp lo activamos cuando necesitamos analizar la composición porcentual de una muestra. Cada "quesito" es un ángulo calculado con precisión ($360^\circ$) que nos muestra de un vistazo quién se está llevando la mayor parte de la tarta y quién se queda con las migas.

💎 TRUCO DE PRECISIÓN: Ideal para visualizar cuotas de mercado, presupuestos o la composición de una audiencia.

Texto

El Diagrama de Sectores (o circular) representa las frecuencias de las categorías como porciones de un círculo completo.

🍕 Anatomía del Sector

Ángulo de Sector: El tamaño de cada porción se calcula con la fórmula: $\alpha = \frac{f_i \cdot 360^\circ}{N}$. A mayor frecuencia, mayor ángulo.

Visión Porcentual: Es la mejor herramienta para visualizar el porcentaje ($h_i \cdot 100$). La suma de todos los sectores siempre debe completar el $100\%$.

Uso Óptimo: Se recomienda para variables con pocas categorías (máximo 5 o 6) para mantener la legibilidad del "escáner".

⚠️ Error Común: No intentes comparar dos diagramas de sectores de diferentes tamaños de muestra ($N$) a ojo; lo que importa aquí es la proporción, no el volumen total.
