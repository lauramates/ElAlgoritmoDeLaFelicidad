Tipos de variables

Definición

Variable estadística: es la característica o propiedad objeto del estudio.

Tipos de variables estadísticas:

  • Cualitativa: si no se puede expresar mediante una cantidad o número.
  • Cuantitativa: Si se expresa mediante un número.

Dentro de las cualitativas podemos distinguir entre:

  • Discreta: solo toma valores aislados.
  • Continua: puede tomar todos los valores de un intervalo.

La obtención de datos puede ser:

  • Directa: se obtienen midiendo, observando o preguntando a los individuos. Mediante cuestionarios se obtiene la información.
  • Indirecta: los datos se obtienen de diversas fuentes como bases de datos, prensa, instituciones, etc.

Ejercicio Resuelto

Ejemplo:

Clasifica las características sobre las que se ha preguntado en un estudio estadístico: edad, altura, peso, nº de hermanos, nacionalidad, asignatura optativa.
